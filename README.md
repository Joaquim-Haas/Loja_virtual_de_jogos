# Loja Virtual de Jogos

Uma loja virtual de jogos via console inspirada na Steam, desenvolvida em Java.

## Funcionalidades

- **Carrinho de compras**
- **Sistema Admin**
- **Controle de saldo**
- **Biblioteca pessoal**

## Foi usado
- **Java 21**
- **ArrayList, HashMap**
- **POO**
- **Exception Treatment**

## Baixe a pasta do código
[️ Download (ZIP)](https://github.com/Joaquim-Haas/Loja_virtual_de_jogos/archive/refs/heads/master.zip)

## Como executar 
```bash
git clone https://github.com/Joaquim-Haas/Loja_virtual_de_jogos.git
cd Loja_virtual_de_jogos/src
javac *.java
java Main
```
