# Loja Virtual de Jogos

Uma loja virtual de jogos via console inspirada na Steam, desenvolvida em Java.

## Funcionalidades

- **Carrinho de compras**
- **Sistema Admin**
- **Controle de saldo**
- **Biblioteca pessoal**

## Foi usado
- **Java 21**
- **ArrayList, HashMap**
- **POO**
- **Exception Treatment**
